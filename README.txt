Розробити систему обліку товарів на конвеєрі. Практичним завдання є розпізнавання та підрахунок товарів, які розміщені на рухомій стрічці виробничого конвеєра. Результати обліку повинні заноситись у віддалену БД , яка  доступна лише по каналу TCP/IP (допускається написання власної простої СУБД, або використання існуючої).  Доступ до СУБД – захищений (метод шифрування – довільний). У БД повинна зберігатись історія роботи багатьох конвеєрів, коже з яких об лаштований розроблюваною системою. У програма повинна існувати можливість перегляду історії обліку (як по окремому конвеєрі, так по усіх).
Вхідна інформація для систему управління окремим конвеєром знімається з відеокамери, або зчитується з avi-файла (зразок відео рухомого конвеєра додається). Уся інтелектуальна обробка і її результати повинні бути синхронізовані із швидкістю руху стрічки конвеєра. 
При вирішенні завдання використати бібліотеку OpenCV. Тип проекту будь-який.
